# Product Manager Self-Assessment

A free, open-source self-assessment tool for Product Managers to identify strengths and growth opportunities across 11 key PM dimensions.

## Take the Assessment

**[Take the assessment online](https://aj1021n-netizen.github.io/pm-self-assessment/)** (no download required)

Or download `index.html` and open it locally in your browser.

## Features

- **11 PM Dimensions** — Decision-Making, Prioritization, Customer Obsession, Execution, Strategic Thinking, Communication, Leadership, Intellectual Humility, Product Strategy, Customer Insight, Managing Up

- **Behavioral Scenarios** — Questions based on what you actually do, not what you think you should do

- **Visual Results** — Radar chart showing your scores across all dimensions

- **Personalized Recommendations** — Curated Lenny's Podcast episode recommendations for your growth areas

- **Privacy-First** — All scoring happens in your browser. Optional anonymous submission for aggregate benchmarks.

## How It Works

1. Answer 48 behavioral scenario questions (15-20 minutes)
2. See your scores across 11 dimensions
3. Get personalized podcast recommendations for growth areas
4. Optionally share anonymous scores to help build PM benchmarks

Based on insights from experts on [Lenny's Podcast](https://www.lennysnewsletter.com/podcast): Shreyas Doshi, Marty Cagan, Teresa Torres, Julie Zhuo, Kim Scott, Gibson Biddle, and more.

## Scoring

- **Emerging** (1.0-1.74) — Starting point, building awareness
- **Building** (1.75-2.49) — Growing skills, gaining consistency
- **Proficient** (2.50-3.24) — Independently effective, reliable patterns
- **Leading** (3.25-4.00) — Coaches others, strategic application

## Self-Hosting

This is a single HTML file with no dependencies (Chart.js and EmailJS loaded via CDN). To self-host:

1. Download `index.html`
2. Host on any static file server (GitHub Pages, Netlify, Vercel, S3, etc.)
3. Optionally update the EmailJS configuration to collect your own aggregate data

## Customization

Feel free to fork and customize:
- Add/modify questions
- Change scoring weights
- Update recommendations
- Rebrand for your organization
